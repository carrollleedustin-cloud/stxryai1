## üöÄ QUICK START GUIDE

### 1. Clone & Install (5 minutes)
```bash
git clone https://github.com/your-username/stxryai.git
cd stxryai
npm install
```

### 2. Configure Environment (10 minutes)
```bash
cp .env.example .env.local
# Edit .env.local with your credentials
```

### 3. Initialize Database (15 minutes)
```bash
# Run SQL scripts in Supabase SQL Editor
psql -f database/init.sql
psql -f database/storage-buckets.sql
```

### 4. Start Development (2 minutes)
```bash
npm run dev
# Open http://localhost:4028
```

### 5. Deploy to Production (30 minutes)
```bash
# Configure GitHub Actions secrets
# Push to main branch
git push origin main
# Netlify will auto-deploy
```

**Total Setup Time:** ~1 hour
